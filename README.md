# Guides to creating Good Enough VR Trainers with Unity and Meta Quest 3

The references are organised into the following categories to progressively get the user to be familiar with Unity and Meta Quest.

## Introduction Packages
The Introduction packages cover how to set up a VR environment from scratch. These form a good foundation and understanding of the platforms.

## Focused Packages
The Focused packages cover short-cuts to using embedded building blocks with Unity to quickly programme specific functions into the VR training.
